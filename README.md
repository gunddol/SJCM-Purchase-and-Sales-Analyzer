# 📖 매입/매출 꼼꼼 분석기 사용 설명서

안녕하세요! 이 프로그램은 복잡한 엑셀 계산을 대신 해주는 똑똑한 도구입니다.  
**매입 자료**와 **매출 자료**만 넣으면, 알아서 계산하고 표와 그래프로 보여줍니다.

어렵지 않으니 천천히 순서대로 따라해 주세요.

---

## 1단계: 매입 파일 넣기 (물건 사온 것)

화면에 **1) 매입 데이터 업로드**라고 적힌 상자가 있습니다.
1. `파일 선택` 버튼을 눌러주세요.
2. 가지고 계신 **매입 엑셀 파일**을 찾아서 열어주세요.
3. 파일이 잘 들어가면 '00행'이라고 글씨가 나옵니다.

## 2단계: 매출 파일 넣기 (물건 판 것)

바로 옆에 **2) 매출 데이터 업로드**라고 적힌 상자가 있습니다.
1. 마찬가지로 `파일 선택` 버튼을 눌러주세요.
2. **매출 엑셀 파일**을 찾아서 열어주세요.
3. 매출 파일은 여러 개여도 괜찮습니다. 파일이 더 있으면 계속 넣으시면 됩니다.

> **참고:** 만약 기존에 넣은 매출에 **이어서 더 추가**하고 싶으시면, `기존 매출에 추가` 체크박스를 누르고 파일을 넣으세요.

## 3단계: 결과 확인하기 (몇 월인지 고르기)

파일을 다 넣으셨으면, 아래쪽 **3) 분석 설정**을 보세요.
1. **'월 선택'** 버튼을 누르시고, 확인하고 싶은 달(예: 2024-01)을 골라주세요.
2. 그러면 자동으로 계산이 끝납니다!
   - **월 매입수량**: 한 달 동안 얼마나 사왔는지
   - **월 매출수량**: 한 달 동안 얼마나 팔았는지
   - **순증감**: 남은 재고가 얼마인지 (초록색이면 남음, 빨간색이면 모자람)

## 4단계: 표와 그래프 보기

화면 아래쪽을 보시면 자세한 내용이 나옵니다.
- **월 전체 추이**: 날짜별로 얼마나 팔렸는지 꺾은선 그래프로 보여줍니다.
- **월 요약 표**: 품목별로 정리가 되어 있습니다. 표를 누르면 더 자세히 볼 수 있습니다.

## 💡 팁: 원하는 품목 찾기
화면 **맨 위**에 긴 검색창이 있습니다.
- 찾고 싶은 **품목 번호**나 **이름**을 입력해보세요.
- 입력하는 즉시 그 물건만 쏙 골라서 보여줍니다.

## 🔄 처음부터 다시 하고 싶을 때
화면 맨 위에 있는 **`초기화`** 버튼을 누르면 모든 내용이 지워지고 처음 상태로 돌아갑니다.
